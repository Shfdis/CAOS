УСЛОВИЕ ЗАДАЧИ
Передача 32-битного целого числа между двумя процессами через сигналы SIGUSR1/SIGUSR2. PID собеседника вводится с клавиатуры. Каждый бит отправляется отдельным сигналом, после каждого бита — подтверждение (ACK) SIGUSR1. Завершение передачи — SIGINT от отправителя.

РЕАЛИЗАЦИЯ
Две программы: sender.c и receiver.c. Используются SIGUSR1/SIGUSR2 для битов, SIGUSR1 как ACK, SIGINT для завершения. Биты идут от младшего к старшему. Формат числа — знаковый int32.

ТЕСТИРОВАНИЕ
Сборка: make
Терминал 1: ./receiver (печатает PID, ввод PID sender)
Терминал 2: ./sender (печатает PID, ввод PID receiver, затем число)
Пример: передано -123, receiver выводит -123.

ВЫВОДЫ
Передача битов через сигналы с подтверждением реализована. Потери нет, порядок PID не важен. Завершение по SIGINT.
